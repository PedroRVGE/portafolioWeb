About Me

I'm a software engineer from Universidad EIA, specializing in automating processes for different applications. I have a strong background in software development, system optimization, and workflow automation. Additionally, I completed a DevOps course from IBM, which has enhanced my expertise in CI/CD pipelines, cloud infrastructure, and containerization. I'm passionate about improving efficiency through automation and continuously learning new technologies to refine development processes.

Projects

Game Store

This project involves the automating of the sales process and stock control of a physical store.

Technologies: Java, Eclipse IDE, Swing code

I designed and implemented the entire product control system for the application, including the user interface, using Java Swing. This system allows for seamless product management, including adding, modifying, and tracking inventory. Additionally, I developed an intuitive user view that enhances the user experience by providing structured navigation and easy interaction with the system. The design follows Object-Oriented Programming principles, ensuring maintainability and scalability.

Library Management

This project involves the automating of the library processes making it easy for the user and the clients to manage lending books and having a control over it.

Technologies: Java, Eclipse IDE, Swing code

I contributed to developing the backend of this library application, implementing Object-Oriented Programming principles for a scalable and maintainable structure. Additionally, I assisted in integrating a small database to manage records efficiently.

Communication tools through VR

This project was created to show a possible application of VR in the communication industry, for this a simulation of a long distance call through VR was developed.

Technologies: Unity, MySQL

I contributed in the development of the VR environment and the connection with the database specifically user register and login.
